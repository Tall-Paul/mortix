What is Mortix.net?

Well, you can view our indiegogo campaign Here for more detail but...

Mortix.net offers transparent, end to end encryption for app developers using a simple API wrapped around PGP

We currently have a simple proof of concept browser app up and running that allows you to send / receive messages. This is not intended to be a 'killer app' for Mortix.net, but rather a a simple demonstration of how the API works

Anything you send via mortix is (as far as anyone currently knows) completely unreadable by anyone other than you Assuming your passphrase is secure

You can register for Mortix here

Please note: due to lack of funds Mortix.net is currently running on a laptop in my front room. If you like the project and want to see it progress to real world use, donate to the campaign!
